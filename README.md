# Ecommerce-Agri-UI

Agri UI Ecommerce is an all in one place for B2B, B2C platform to sell their products.
Live Hosted Link - **[Agri Ecommerce App](https://agri-ecom.netlify.app/)**

## Technology Used

- HTML
- CSS
- JavaScript

## Features

- Login
- SignUp
- Product Listing
- Cart Management
- Catagory wise products
